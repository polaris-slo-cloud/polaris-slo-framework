Class PolarisQueryBase<T>

Common superclass that can be used for implementing PolarisQuery realizations.

Since Polaris queries are supposed to be used as a fluent API, PolarisQueryBase is designed to establish a chain of PolarisQuery instances, with each query referring to its predecessor.
