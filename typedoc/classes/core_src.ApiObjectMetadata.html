Class ApiObjectMetadata

Provides metadata about an ApiObject.

This class is based on V1ObjectMeta of Kubernetes and includes a selection of its properties. Most of the docs were taken from https://github.com/kubernetes-client/javascript/blob/master/src/gen/model/v1ObjectMeta.ts
