Class DefaultComposedMetricMappingWatchEventsHandler

Receives watch events for a ComposedMetricMapping and communicates them to the ComposedMetricsManager.

If supportedSloTargetKinds is configured on the ComposedMetricComputationConfig, only mappings with matching SloTarget types will be passed to the ComposedMetricsManager.
