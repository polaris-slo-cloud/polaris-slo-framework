registerObjectKind

  • registerObjectKind<T>(kind: ObjectKind, polarisType: PolarisConstructor<T>, transformer?: PolarisTransformer<T, any>, config?: PolarisTransformationConfig): void
    • Defined in libs/core/src/lib/transformation/public/common/polaris-transformation-service-manager.ts:45

    Associates the specified object kind with a Polaris type and optionally also with a transformer.

    This allows the PolarisTransformationService to automatically instantiate the appropriate class for a raw orchestrator object, based on the kind information that it provides.

    note

    A transformer registration for a class A applies only to direct instances of A, not to instances of any of its subclasses.

    Type Parameters

    • T

    Parameters

    • kind: ObjectKind

      The ObjectKind that should be registered.

    • polarisType: PolarisConstructor<T>

      The Polaris type to be associated with the object kind.

    • Optional transformer: PolarisTransformer<T, any>

      The PolarisTransformer for the type.

    • Optional config: PolarisTransformationConfig

      (optional) Additional configuration for registration of the PolarisTransformer.

    Returns void
