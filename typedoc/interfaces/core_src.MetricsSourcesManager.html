addComposedMetricSourceFactory

  • addComposedMetricSourceFactory(factory: ComposedMetricSourceFactory<ComposedMetricType<any, any>, any, ComposedMetricParams>, sloTargetType?: ObjectKind): void
    • Defined in libs/core/src/lib/metrics/public/metrics-sources-manager.ts:40

    Adds the specified ComposedMetricSourceFactory to this manager for a particular type of SloTarget.

    Adding the same factory for multiple SloTarget types is possible.

    If a factory is registered without the sloTargetType parameter, it is used within the ComposedMetricType for all SloTarget types, for which no factory has been registered.

    Parameters

    • factory: ComposedMetricSourceFactory<ComposedMetricType<any, any>, any, ComposedMetricParams>

      The ComposedMetricSourceFactory that should be added.

    • Optional sloTargetType: ObjectKind

      The type of SloTarget that this factory should be registered for.

    Returns void
