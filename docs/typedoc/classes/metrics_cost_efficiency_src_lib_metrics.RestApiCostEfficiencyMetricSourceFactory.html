Class RestApiCostEfficiencyMetricSourceFactory

Factory for creating RestApiCostEfficiencyMetricSource instances that supply metrics of type CostEfficiencyMetric.

This factory supports SloTarget types that match ReplicableSloTarget.

Static supportedSloTargetTypes

supportedSloTargetTypes: ObjectKind[] = ...
  • Defined in libs/metrics/cost-efficiency/src/lib/metrics/rest-api-cost-efficiency/rest-api-cost-efficiency-metric-source.factory.ts:22

The list of supported SloTarget types.

This list can be used for registering an instance of this factory for each supported SloTarget type with the MetricsSourcesManager. This registration must be done if the metric source should execute in the current process, i.e., metric source instances can be requested through MetricSource.getComposedMetricSource().

When creating a composed metric controller, the list of compatible SloTarget types is determined by the ComposedMetricMapping type.
