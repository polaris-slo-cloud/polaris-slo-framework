Interface GenericComposedMetricSourceFactory

A GenericComposedMetricSourceFactoryis used to create a ComposedMetricSource instance for a generic composed metric type, scoped to a particular SloTarget.

The difference between this factory type to a ComposedMetricSourceFactory is that a ComposedMetricSourceFactory creates sources for one specific ComposedMetricType, while GenericComposedMetricSourceFactory creates sources for any ComposedMetricType and used as a fallback when no specific factory is registered for a composed metric type.
